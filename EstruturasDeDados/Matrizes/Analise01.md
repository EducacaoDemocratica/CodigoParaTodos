# Exercício 1

[**Ver Algoritmo**](Algoritmo01.md)

*Faça um pseudocódigo que armazene os valores de 1 a 9 em uma matriz M 3x3 e apresente esses valores.*

**Entrada**

- Nenhuma.

**Saída**

- Os números de 1 a 9 em sequência.

**Restrições**

- Nenhuma.

**Exemplo**
| Entrada | Saída         |
| ------- | ------------- |
|         | 1, 2, 3       |
|         | 4, 5, 6       |
|         | 7, 8, 9       |
