# Exercício 30

[**Ver Algoritmo**](Algoritmo30.md)

*Faça um pseudocódigo que informe quantos algarismos primos têm um número inteiro informado pelo usuário.*

**Entrada:**
- Um número inteiro n.

**Saída:**
- A quantidade de algarismos primos em n.

**Restrições:**
- Nenhuma.

**Exemplo:**

| Entrada | Saída |
| ------- | ----- |
| -1519 | 1 algarismos |
| 2357 | 4 algarismos |
| 4914 | 0 algarismos |
