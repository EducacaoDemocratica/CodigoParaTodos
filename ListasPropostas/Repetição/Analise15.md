# Exercício 15

[**Ver Algoritmo**](Algoritmo15.md)

*Faça um pseudocódigo que informe o M.M.C (mínimo múltiplo comum) de 3
números inteiros positivos.*

**Entrada:**
- Três números inteiros m, n e o.

**Saída:**
- MMC de m, n e o.

**Restrições:**
- m, n e o precisam ser positivos.

**Exemplo:**

| Entrada    | Saída   |
| ---------- | ------- |
| 1, 25, 85  | 425     |
| 4, 64, 94  | 3008    |
| 2, 3, 42   | 42      |
