# Exercício 29

[**Ver Algoritmo**](Algoritmo29.md)

*Faça um pseudocódigo que informe a quantidade de algarismos existentes em um número inteiro.*

**Entrada:**
- Um número inteiro n.

**Saída:**
- A quantidade de algarismos em n.

**Restrições:**
- Nenhuma.

**Exemplo:**

| Entrada | Saída |
| ------- | ----- |
| -1509 | 4 algarismos |
| 101 | 3 algarismos |
| 0 | 1 algarismo |
