# Exercício 14

[**Ver Algoritmo**](Algoritmo14.md)

*Faça um pseudocódigo que informe o M.M.C (mínimo múltiplo comum) de 2
números inteiros positivos.*

**Entrada:**
- Dois números inteiros n e m.

**Saída:**
- MMC de n e m.

**Restrições:**
- n e m precisam ser positivos.

**Exemplo:**

| Entrada | Saída |
| ------- | ----- |
| 34, 41  | 1394  |
| 21, 64  | 642   |
| 82, 75  | 7574  |
