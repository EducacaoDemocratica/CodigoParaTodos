# Exercício 17
[**Ver Algoritmo**](Algoritmo17.md)

**Entrada**

- \( n \) número(s) positivo(s).

**Saída**

- A soma dos \( n \) números.

**Restrições**

- O algoritmo deve se encerrar quando -1 for inserido.

**Exemplo**
| Entrada                    | Saída |
|----------------------------|-------|
| 2, -10, 3, 8, 12, -41, 34, -1 | 8    |
| 10, -12, 8, -1              | 6    |
| -1                         | 0    |
