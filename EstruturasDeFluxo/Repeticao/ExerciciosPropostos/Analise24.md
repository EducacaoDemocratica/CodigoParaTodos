# Exercício 24
[**Ver Algoritmo**](Algoritmo24.md)

*Faça um pseudocódigo que informe o valor S dado pela série:*

$$\
S = 1 + 2 + 3 + \ldots + n
\$$

**Entrada**

- Um número inteiro n.

**Saída**

- O valor de S.

**Restrições**

- n precisa ser positivo.

**Exemplo**
| Entrada | Saída |
|---------|-------|
| 15      | 120   |
| 20      | 210   |
| 0       | N deve ser maior que zero. Insira outro número: |
