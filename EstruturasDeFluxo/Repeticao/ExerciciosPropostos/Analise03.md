# Exercício 03
[**Ver Algoritmo**](Algoritmo03.md)

Faça um pseudocódigo para informar a soma dos números inteiros de 1 até 100.

**Entrada**

- Nenhuma.

**Saída**

- Soma dos números de 1 a 100.

**Restrições**

- Nenhuma.

**Exemplo**
| Entrada | Saída |
|---------|-------|
|         | 5050  |
