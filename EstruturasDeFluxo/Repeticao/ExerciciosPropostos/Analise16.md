# Exercício 16
[**Ver Algoritmo**](Algoritmo16.md)

**Entrada**
\( n \) número(s) positivo(s).

**Saída**
A soma dos \( n \) números.

**Restrições**
O algoritmo deve se encerrar quando um número negativo for inserido.

**Exemplo**
| Entrada                           | Saída |
|-----------------------------------|-------|
| 1, 2, 3, 4, 6, 8, 10, 10, 22, 12, -10 | 78   |
| 0, 2, -4                            | -42  |
| -10                               | 0    |
