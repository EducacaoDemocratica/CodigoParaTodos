# Exercício 22
[**Ver Algoritmo**](Algoritmo22.md)

**Entrada**
Números inteiros. O algoritmo encerra quando -1 for inserido.

**Saída**
O maior número inserido no conjunto.

**Restrições**
Nenhuma.

**Exemplo**
| Entrada                       | Saída |
|-------------------------------|-------|
| 1, -10, -9, -8, -7, -11, -1   | -7    |
| -12, 8, 90, -100, -190, -1     | 90    |
| -1                            | Nenhum número inteiro foi inserido. |
