# Exercício 18

*Faça um pseudocódigo que leia um valor qualquer de \(x\) e imprima o valor de \(y\), sabendo que:*

\[y = f(x)\]

\[
\begin{cases}
1, & \text{se } x \leq 1 \\
2, & \text{se } 1 < x \leq 2 \\
x^2, & \text{se } 2 < x \leq 3 \\
x^3, & \text{se } x > 3
\end{cases}
\]

**Entrada**

- Um número \(x\) qualquer.

**Saída**

- \(y\) ou \(f(x)\).

**Restrições**

- Nenhuma.

**Exemplos**

| Entrada | Saída |
|---------|-------|
| 0.222   | 1     |
| 4       | 64    |
| 2.5     | 6.25  |
